<h1 align="center">🧑‍💻 Sufiyan Khan Portfolio</h1>

Welcome to my interactive developer portfolio! This project showcases my skills, projects, and experience using a modern, animated, and terminal-inspired interface.

## 🚀 Features

- **Animated Hero & Navbar**: Smooth transitions and glassmorphism effects
- **Interactive Terminal**: Type commands like `about`, `experience`, `skills`, `projects`, `education`, `contact` for a unique experience
- **Dynamic Sections**: Experience, Education, Projects, and Skills with engaging visuals
- **Particle & Glow Effects**: Modern UI with custom cursor and animated backgrounds
- **Preloader**: Sleek loading animation for first impressions
- **Mobile Responsive**: Looks great on all devices

## 🛠️ Tech Stack

- [Next.js](https://nextjs.org/) (App Router)
- [React](https://react.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [Lucide Icons](https://lucide.dev/)
- [Leaflet.js](https://leafletjs.com/) (for map visualizations)

## 📦 Getting Started

Clone the repo and install dependencies:

```bash
git clone https://github.com/YOUR-USERNAME/YOUR-REPO.git
cd YOUR-REPO
npm install
```

Run the development server:

```bash
npm run dev
# or

```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## 🌐 Deployment

Deploy instantly on [Vercel](https://vercel.com/):

1. Push your code to GitHub
2. Go to [vercel.com](https://vercel.com/) and import your repo
3. Click **Deploy**
